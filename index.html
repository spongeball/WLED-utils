WLED Preset Editor

Standalone browser-based editor for presets.json

Load a presets.json exported from your WLED device, edit it visually, and export it back. No installation, no server — everything runs locally in the browser.

You can edit every field defined in the WLED JSON API: name, brightness, transition, on/off state, and all segment parameters including effect, speed, intensity, palette, colors, LED range, grouping, CCT, and custom sliders. Select multiple presets with the checkboxes to apply bulk changes across all of them at once.

Open Editor Open Remote

How to use

  • Open the editor and load your presets.json via the button or drag and drop
  • Click a preset row to open it in the editor
  • Tick the checkbox on the left of each row to add presets to the bulk selection
  • Shift-click a row to select a range of checkboxes
  • Export when done and upload the file back to your device via WLED's Backup/Restore page
